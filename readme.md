AUTHPROJECT
Simple système d'authentification.
Il ne s'agit pas d'un projet orienté objet mais de code PHP procédural.

	Ce système gère:

* Inscription avec token de validation (avec expiration)
* Connection avec user ou email
* Système de sauvgarde mot de passe (sécurisé)
* Système d'oublie de mot passe
* Changement de mot de passe possible une fois connecté
